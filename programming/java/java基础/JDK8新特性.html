# Lambda 表达式

    return b.compareTo(a);
}

转换为Lambda表达式为：
        
(a, b) -> b.compareTo(a)

# 1、foreach如何提前终止

普通的for循环，终止循环使用break，中断本次继续下次循环使用continue，如果使用return则终止此for循环和整个方法

java8的foreach中，break和continue不被允许使用；关键字return，在这里执行的效果相当于普通for循环里的关键词continue

  • java8的foreach中，要提前终止的方法：
    • 抛出异常然后在循环外套一层try..cache 或者是 改成原始的for循环

# 函数式接口

# 1、什么是函数式接口

  • 只包含一个抽象方法的接口
  • 可以通过Lambda表达式来创建该接口的对象
  • 可以在任意函数式接口上使用@FunctionalInterface注解

# 案例 Runnable

@FunctionalInterface注解修饰；Runnable接口就是一个函数式接口

@FunctionalInterface
public interface Runnable {
    public abstract void run();
}

# 2、常用四大核心函数式接口

  • Consumer<T>
    • 消费型接口
    • 对类型为T的对象应用操作
  • Supplier<T>
    • 供给型接口
    • 返回类型为T的对象
  • Function<T, R>
    • 函数型接口
    • 对类型为T的对象应用操作，并返回结果，结果为R类型的对象
  • Predicate<T>
    • 断定型接口
    • 确定类型为T的对象是否满足某约束，并返回boolean值

# Stream

  • java.util.stream
  • 能用声明式的方式处理数据（集合、数组等）
  • Stream流支持同步执行也支持并发执行
    • stream()创建串行流
    • parallelStream() 创建并行流
  • 只能对实现了java.util.Collection接口的类做流的处理操作

# 1、获取流的方式

  • 对于实现了java.util.Collection接口的类：直接.stream()即可
  • 对于实现了java.util.Map接口的类：需要keySet().stream()或者values().stream()或者entrySet().stream()

# 2、常用方法

  • Filter
    • 过滤掉不符合条件的
public static void main(String[] args) {
    List<Integer> list = new ArrayList<>();
    list.add(12);
    list.add(34);
    list.add(23);
    list.add(11);
    list.add(17);

    //  过滤掉集合中奇数
    list.stream()
            .filter(item -> item % 2 == 0)
            .forEach(System.out::println);
    // 输出：
    // 12
    // 34
}

# 脐风

新生儿脐部是一个细菌容易繁殖的地方，如果护理不当，宝宝很容易感染，导致脐风，严重时还会引起菌血症和败血症。

# 1、什么是脐风

由于接生人员的手或所用的剪刀、纱布未经消毒或消毒不严格，或结扎不紧，风冷水湿从脐部入侵婴儿身体，或婴儿洗浴时牵动脐带、使脐带脱落过早，局部受伤感染所致

# 2、病症

初发病时： 频频喷嚏、啼哭不安、精神躁扰不安，吮乳口松

发作期： 身法寒热、脐肿腹胀、牙关经闭、不能吮乳、唇口收缩、口禁舌僵、欧吐白沫、啼不出声

严重时： 颜面，口唇，四肢抽搐、颈项僵直，角弓反张等症状（角弓反张：项背高度强直，使身体仰曲如弓状）

# 3、如何防治护理

脱落前：

  • 在新生儿脐带脱落前，最好每日检查脐部，观察脐带残端有误出血、渗血、渗液等情况

    • 若发现出血应该及时送医院处理

    • 一般情况使用消毒棉签蘸75%的酒精涂擦脐部，由内向外做环形消毒，然后盖上消毒纱布，再用胶布固定，以防止感染

  • 勤换尿布/尿不湿，并且避免尿布/尿不湿直接覆盖在脐部上

  • 给宝宝洗澡时，尽量不要打湿脐部，更不能将宝宝全身浸在澡盆内，以防脐部糜烂处水浸湿而引起感染

脱落后：

  • 脐窝稍潮湿，每天用75%的酒精擦洗，每日2~3次，知道局部红肿消退、干燥

  • 注意观察创面，如有液体分泌物流出，或红肿表现，且咳嗽哭闹严重时，应及时送医

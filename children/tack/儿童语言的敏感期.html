# 儿童语言的敏感期

  • 孩子所掌握的语言和词汇不会凭空跳出来的，会受他们所处环境的限制

  • 处于语言敏感期的孩子并不代表他们什么都能马上学会，不能拔苗助长地要求孩子背诵唐诗之类的东西。我们只需要提供给孩子充足的锻炼机会和语言要素就可以了。过早的接触知识型的东西，对他们的语言学习并不一定有好处。

  • 这个阶段的孩子理解能力有限，在游戏或沟通中，家长不能着急，不能强迫孩子，否则会给孩子造成心理阴影

# 一、出生后十个月内的语言发展规律

  虽然1岁以内的孩子还不会说话，但是并不代表他们没有在为自己的语言发展做准备

对于1岁以前的孩子来说，他们会经历两个重要的阶段：

  • 1、他们无意识地学习了语言，并逐渐把这种无意识向有意识转变（接收大人的语言）

  • 2、他们开始使用语言，哪怕这种语言还仅仅是简单的发音（如爸爸，妈妈）

# 二、一岁左右儿童的语言发展规律

  1岁左右，孩子开始有意识地说话，这个时候他们依然无法清晰地表达自己的想法，甚至发音都很模糊，但是他们的语言已经有了明确的目的

  1岁左右的孩子爱发火，原因就在与沟通不畅，孩子说不明白，着急上火（当一个人无法和别人正常交流时，他们难免会焦急甚至恼怒，1-2岁的孩子身上也会出现同样的现象），这种情况家长不必大惊小怪。

重点： 当孩子因为说不明白而发火时，家长不能去绞尽脑汁地想孩子为什么生气，去提前猜出孩子的需求，并及时满足孩子，这会导致孩子说话很晚。我们要让孩子潜意识里有表达的欲望，这才有利于他们及早开口说话

# 三、2岁以内孩子的语言训练方法

  • 小游戏
  • 多与孩子说话

# 小游戏：

1、2-6个月的孩子

【与父母一起笑，锻炼孩子的面部肌肉，丰富孩子的表情，培养孩子开朗的性格】

  • 将孩子抱在怀里，与孩子轻声聊天，吸引她的注意力
  • 开始慢慢地对孩子做各种表情，吸引孩子与家长一起做（如张嘴，瞪眼，笑，哭等多种表情，一直到孩子开始高兴的学习家长的表情为止）

2、6个月以上的孩子

【培养孩子的语言意识和语言运用能力；锻炼孩子的语言发展能力】

  • 父母一方将孩子抱在怀里，面朝将要外出的爸爸（妈妈）；将要外出的爸爸（妈妈）要朝孩子摆手说：“再见，宝贝”。同时，抱着孩子的家长可以用手拿起孩子的手，模仿对方的动作摆手，引导孩子说“再见，爸爸（妈妈）”
  • 等孩子能够主动说“再见”后，每当家人外出，都要提醒孩子说再见。要逐渐把孩子说“再见”的对象扩展到外人如邻居或者其它小朋友

3、1.5岁-2岁的孩子 【锻炼孩子的语言功能，思维组织能力】

  • 准备一些画有小动物的卡片，和孩子一起认识动物，然后逐个向孩子介绍这些小动物的叫声，引导孩子跟着一起学动物的叫声（每次教给孩子的动物叫声不能太多，不能超过5种，否则会影响孩子的记忆效果）
    • 注意：在教学过程中，孩子可能会胡编乱造并乐此不疲（如“创造”一些动物的叫声，如蛇是“哇哇”大叫的，孩子还一边强调自己的意见还一边哈哈大笑。这个时候家长不必批评孩子，这是他们创造性思维极大发挥的时候，我们不要和孩子对着干，转移话题就好）
  • 给孩子讲睡前故事（搭配故事的绘本）
    • 选择合适的故事，这个年龄段的不需要复杂的故事，故事有节奏和趣味就行
    • 讲故事的时候要多重复，孩子往往会对某一细节非常感兴趣。针对这种“兴趣点”，我们就应该多重复，可以尝试对细节进行细微的改动，留下想象的空间。培养孩子的纵向思维发育
    • 要注意讲故事的时间，长短。最好是悬在孩子醒来和晚上入睡前这两个时间段，时间掌握在几分钟以内，在孩子不耐烦前结束（2岁以下的孩子很难保持长时间的注意力）
  • 增加孩子词组的使用
    • 如：给孩子一个小熊玩具，孩子很容易说出“熊”这个词，我们可以引导说“宝宝，我想要小熊，要-小-熊”
  • 《视觉大发现》可以锻炼孩子的视觉能力。这个阶段的孩子刚刚开始能够分辨上下左右等方位词，通过这个游戏可以让孩子的空间思维得到进一步发展，相应语言表达能力也会提高
  • 看图说话：是对一个孩子的思维和语言能力锻炼效果极强的训练方式，组图则是对孩子的逻辑思维能力和语言组织能力有所锻炼

4、2岁的孩子 【锻炼孩子的语言信息接收能力，记忆能力，语言组织能力】

  • 背诵儿歌
    • 准备儿歌，读给孩子听。当孩子听懂并弄清内容后，逐字逐句教给孩子，并一起背诵
  • 复述和转述一句话（刚开始可以让孩子复述我们的话（简短一些的），当孩子能够完整重复长句后，可以让孩子充当传话筒角色）
